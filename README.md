## ğŸ¯ Project Overview

**MealLense** is an innovative AI-powered mobile application that transforms how users interact with food. By leveraging state-of-the-art computer vision and deep learning models, MealLense enables users to instantly identify meals through image recognition and receive comprehensive recipe information, nutritional analysis, and cooking instructions.

### ğŸ¯ Problem Statement
In today's fast-paced world, many individuals struggle to identify unfamiliar dishes, understand their components, and access reliable cooking information efficiently. Traditional text-based recipe searches are time-consuming and often fail to capture the visual nature of food discovery.

### ğŸ’¡ Our Solution
MealLense bridges this gap by offering:
- **Instant meal identification** from photos
- **Comprehensive recipe details** including ingredients and step-by-step instructions
- **Nutritional analysis** and allergen detection
- **Personalized recommendations** based on user preferences
- **Multi-cuisine support** with focus on Arabic and international dishes

## ğŸš€ Installation & Setup

### Prerequisites
- Flutter SDK 3.35.7 or higher
- Dart 3.9.2 or higher
- Android Studio / VS Code
- Git
- Firebase account
- Azure account (for Custom Vision)

### Step-by-Step Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/layanbuirat/Graduation-Project-MealLense-Mobile-App-with-Website-Development-FAQ-Message-management-.git
   cd Graduation-Project-MealLense-Mobile-App-with-Website-Development-FAQ-Message-management-
   ```

2. **Install Dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY=your_gemini_api_key
   AZURE_PREDICTION_KEY=your_azure_key
   AZURE_ENDPOINT=your_azure_endpoint
   SPOONACULAR_API_KEY=your_spoonacular_key
   EDAMAM_APP_ID=your_edamam_id
   EDAMAM_APP_KEY=your_edamam_key
   ```

4. **Firebase Setup**
   ```bash
   flutterfire configure
   ```

5. **Run the Application**
   ```bash
   flutter run
   ```
